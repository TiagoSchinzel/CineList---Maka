 CineList: A Personal & Aesthetic Movie Tracker
CineList is a cozy, custom-built web application designed to help you organize your cinematic journey. Created with a soft, "coquette" aesthetic in mind, it combines functionality with a delightful user experience featuring pastel pink tones and cute cat-inspired details. 🐾✨
✨ Key Features

    Smart Watchlist: Keep track of movies and series you want to watch and those you've already enjoyed.
    Google Calendar Integration: Never miss a premiere! Schedule upcoming releases directly to your Google Calendar with a single click.
    Aesthetic Dashboard: A clean, responsive interface with rounded corners, "light pink" themes, and subtle kitty patterns.
    Progress Tracking: Easily move titles between "To-Watch" and "Watched" lists to keep your entertainment life organized.

🛠️ Tech Stack

    HTML5 & CSS3 (Custom Flexbox/Grid layouts)
    JavaScript (Dynamic event handling and Calendar API integration)
    Google Calendar API (Deep linking for scheduling)

Built with ❤️ for a special friend who loves cinema and cats.
